# OM3D
EPITA course for 5th year students

MOOC: https://moodle.epita.fr/course/view.php?id=2292


### How to build
Requirements: cmake 3.20 minimum, C++17, and OpenGL 4.5.
```bash
# At the project root
mkdir -p build/debug
cd build/debug
cmake ../..
make
```
